# Bomberman  
## Introduction:  
A two-player bomberman game  
## Control:  
Player1: Move: W/A/S/D, Attack: Space  
Player2: Move: Up/Down/Left/Right arrow, Attack: Enter  
## Platform:  
Unity 2018.3.7f1
